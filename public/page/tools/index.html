Tools

On this page, we will provide links to code repositories for different methods used in our projects. We also share some interactive software tools for generating time series from dynamical systems.

  • Wrapper functions in R that simplify lagged CCM analyses with surrogate testing. These functions accompany our study of the forcing of Late Pleistocene ice volume by spatially variable summer energy.

  • An interactive Daisyworld model that can be used to generate time series of components of a coupled dynamical system involving external forcing and internal feedbacks.

  • Coming soon: a code repository of R functions running Julia under the hood for computing transfer entropy using a stationary Markov approximation method - stay tuned!
